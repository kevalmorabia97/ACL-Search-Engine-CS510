# Academic-Paper-Search-Engine-CS510

This Search Engine uses BM25 algorithm to rank documents according to the query.

To run the application:
```
python3 run.py
```

This code has been tested to work on the following dependencies:
```
pandas==0.23.4
rake_nltk==1.0.4
Flask==1.1.1
rank_bm25==0.2
nltk==3.4.5
Flask_Cors==3.0.8
```
